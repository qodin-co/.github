## 📦 First Package

### [@qodin-co/sol](https://github.com/qodin-co/sol)

**Solar-aware React widgets that follow the real position of the sun.**

> Dark mode reacts to a preference. Sol reacts to place and time.

Sol computes the sun's actual position from your user's location and time,
then transitions the UI through **9 solar phases** — from dawn through midnight
— with 10 skins, optional live weather, and animated blends between phases.

- Zero API keys — solar math runs locally via NOAA equations
- No geolocation permission required — falls back to browser timezone
- SSR-safe — Next.js, Remix, TanStack Start, Blade, Vite

```bash
bun add @qodin-co/sol
# or
npm install @qodin-co/sol
```

→ [Explore Sol](https://github.com/qodin-co/sol)
